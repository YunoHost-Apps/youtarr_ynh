Install this app at the root of a domain or subdomain (e.g. `youtarr.example.com`), not under a path prefix (e.g. `example.com/youtarr`): Youtarr's web UI references its static assets with absolute root paths, so it won't load correctly under a subpath.

Make sure you have enough free disk space for downloaded videos: choose a storage path with plenty of room when the install asks for one, or leave the default (the app's data directory) if you plan to keep downloads small.
